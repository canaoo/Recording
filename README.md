# Keyboard Recordings

 キーボードでの演奏・録音をして投稿することができるアプリケーションです。<br>
 投稿する際は、投稿フォームに曲名を入力し録音したwavファイルを指定します。<br>
 ユーザが投稿した録音一覧を表示したり、検索することができます。<br>
